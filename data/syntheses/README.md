# Polished synthesis outputs

This directory holds LLM-polished prose for each lens, generated by
`pnpm synthesize:polish` (curator-side; uses your ANTHROPIC_API_KEY).

Files committed here are read by `/lens/[id]/synthesis` at build time
and shipped as part of the static export. The deployed page shows the
polished version by default with a "curator" toggle to drop back to
the source-grounded deterministic outline.

Workflow:
  1. ANTHROPIC_API_KEY=sk-ant-... pnpm synthesize:polish
  2. Review each *.md output for factual drift / missing citations
  3. git add data/syntheses/*.md && git commit && git push
  4. Vercel auto-deploys with polished prose baked in

Provenance: input is curator-authored fields (brief, master_summary,
lens descriptions). Output is paraphrase-of-paraphrase, tagged
`llm-rendered-prose` per spec §5. Curator review is REQUIRED before
publish — the system prompt constrains the LLM to no-new-claims and
preserves citations, but verification is on the curator.

Cost: ~$0.30–$1.00 per full pass (13 lenses) using Sonnet 4.6.
